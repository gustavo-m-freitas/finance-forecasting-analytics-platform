# Module 6 — Streamlit Dashboard

## Overview

This module implements the interactive application layer of the platform using Streamlit.

The objective is to transform the analytical outputs generated in the previous modules into a fully interactive web application, enabling dynamic exploration of financial data, operational drivers, and forecasting results.

Unlike the BI layer (Module 5), which focuses on executive dashboards, this module provides a hands-on analytical interface where users can apply filters, navigate across views, and interact with the data in real time.

The application connects directly to the Financial Data Warehouse and integrates outputs from the automation pipeline and forecasting models.

## Key Components

- Streamlit-based web application  
- Interactive financial dashboards  
- Real-time data access via PostgreSQL  
- Dynamic filtering and navigation  
- Plotly visualizations  
- Forecast vs Actual comparison  
- Scenario-based financial analysis  

---

## Dashboard Perspectives

The application provides three main analytical views:

**Financial Performance Journey**

Interactive overview of revenue, profitability, and financial performance across time, including KPI indicators and dynamic time filtering.

**Revenue Drivers Analysis**

Breakdown of revenue by product lines and sales channels, including price vs volume decomposition and operational performance analysis.

**Forecast & Scenario Planning**

Interactive visualization of forecasting results, including scenario analysis, variance tracking, and comparison between actual and projected performance.
