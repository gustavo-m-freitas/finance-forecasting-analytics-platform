# Module 5 — Executive BI Dashboard

## Overview

This module implements the Business Intelligence (BI) layer of the platform using Power BI.

The objective is to transform the analytical outputs generated in the previous modules into interactive dashboards that support financial monitoring, forecasting interpretation, and executive decision-making.

The dashboards connect directly to the analytical datasets produced by the Financial Data Warehouse and the forecasting models.

## Key Components

- Power BI financial dashboards
- Visualization of IFRS financial statements
- Revenue and performance trend analysis
- Forecast vs Actual comparison
- Scenario-based financial insights

---

## Dashboard Perspectives

The BI layer provides three main analytical views:

**Financial Performance Journey**

High-level overview of revenue, profitability, and financial performance across time.

**Revenue Drivers Analysis**

Breakdown of revenue by product lines and sales channels to understand the main drivers of financial performance.

**Forecast & Scenario Planning**

Visualization of forecasting results and scenario analysis generated in the forecasting module.

---

## Role in the Platform

Module 5 represents the **executive visualization layer** of the platform.

It translates the outputs of the data warehouse, automation pipeline, and forecasting models into interactive dashboards that support financial analysis and strategic decision-making.
