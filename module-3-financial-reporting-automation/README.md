# Module 3 — Financial Reporting Automation

## Overview

This module implements the **Python automation pipeline** used to extract, transform, and prepare financial data from the **Financial Data Warehouse**.

The goal is to replace manual financial reporting with **reproducible analytical workflows**.

## Key Components

- **SQLAlchemy** database connection to PostgreSQL
- **Pandas** data extraction and transformation
- Automated **financial KPI calculations**
- **Data validation checks**
- Standardized **reporting datasets**

## Role in the Platform

This module connects the **Data Warehouse** with the **analytics and forecasting layers**.
